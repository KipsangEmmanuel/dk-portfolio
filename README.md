## About

This portfolio serves as a showcase of my professional skills, projects, and contact methods. The website emphasizes **clean design, interactive elements, and modern UI patterns**. Key sections include:

- **Hero Section:** Introduction, call-to-action buttons for projects, contact, and CV download.
- **About Section:** My approach to software engineering and system design.
- **Technologies:** Tools and tech stack I specialize in, with concave card UI for visual emphasis.
- **Contact Section:** Direct email form, GitHub, WhatsApp, and call links.
- **Footer:** Social links, resume download, and consistent theme with dark/light mode.

---

## Features

- **Responsive Layout:** Works across desktop, tablet, and mobile screens.
- **Dark/Light Mode:** Toggle between dark and light themes with CSS variables.
- **Concave UI Effect:** 3D-style concave cards for input fields, buttons, and tech cards.
- **Interactive Components:**
  - Hover effects on buttons and cards.
  - Scroll spy navigation highlighting active sections.
- **Download CV:** Direct link to download resume.
- **Contact Form:** Sends email directly using `mailto:` with prefilled subject and body.
- **Social & Direct Links:** GitHub, WhatsApp, and phone call links.
- **Technologies Icons:** React, Node.js, Express.js, Python, Django, TailwindCSS, PostgreSQL, Docker, and more.

---

## Technologies Used

- **Frontend:** React, TailwindCSS
- **Icons:** React Icons (SiReact, SiNodedotjs, etc.)
- **Styling:** CSS Variables, concave UI effects
- **Deployment:** Static HTML output via Vite or Next.js
- **Optional Backend:** For form handling or dynamic content

## Getting Started

### Prerequisites

- Node.js >= 18
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Kibetkipsang/portfolio.git
cd portfolio

# Install dependencies
npm install
# or
yarn install

# Run locally
npm run dev
# or
yarn dev
```
